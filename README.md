# Copilot-generated-Linux-and-PowerShell-scripts
Building a Script toolbox to be used on Linux terminals and MS PowerShell with project ideas from my home systems. Start date February 2026

These are created testing the use of Copilot using MS Edge on Microsoft ( eventualy desk top Copilot ) and Linux ( Mint currently, addedWSL on the Windows ) platforms and allowing Copilot to increase the build structures incrementaly as I progress.

Since AI often can create some errors in the usage at times, please take care and test these at your own peril... there seems to be some differences in the PowerShell implementation in Linux and Windows right now but it may be just configurations issues.

I used to build a lot of automation software in my professional career many years ago working for United Technologies Automotive... and now often work as a voluntary admin on social media sites.
